# minimal6

Minimal Hyprland dotfiles.

## Setup


`git clone https://github.com/fallenwesii/minimal6.git`
`chmod +x setup.sh && ./setup.sh`

Installs packages (pacman + AUR), links configs, generates themes, and applies Gruvbox.

## Keybinds

Press `Super + H` in-session for the full cheatsheet.
