# minimal6

Minimal Hyprland dotfiles.

## Setup

```bash
git clone https://github.com/fallenwesii/minimal6.git
chmod +x setup.sh && ./setup.sh
```

Installs packages (pacman + AUR), links configs, generates themes, and applies Gruvbox.

## Uninstall

To undo the symlinks, scripts, and theme overrides set up by minimal6 (without uninstalling any applications):

```bash
./uninstall.sh
```

## Keybinds

Press `Super + H` in hyprland session to view all keybindings.
